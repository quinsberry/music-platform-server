Music platform backend part. </br>

**Technologies:** NodeJS, MongoDB, NestJS, Typescript </br>

## Installation

1. `npm install`
2. Install MongoDB if you didn't do that yet.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
